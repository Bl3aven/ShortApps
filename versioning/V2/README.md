# ShortApps V2

Base retenue pour la suite du projet.

* `ShortApps-V2-win32-x64.zip` : package Windows complet.
* `cahier-des-charges-v2.md` : cahier des charges corrige correspondant a cette version.

## Lancement

Extraire le ZIP sous Windows, puis lancer :

```text
ShortApps.exe
```

Il faut conserver tout le dossier extrait. Le fichier `.exe` seul ne suffit pas.
